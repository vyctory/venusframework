Les configurations

La manipulation des configuration peut-être très utile dans l'avancée de vos projets. Vous pouvez par exemple indiquer

à votre fichier de configuration qu'il doit faire référence à un autre fichier de configuration et pas à lui même : 

{
    "redirect": "Helium"
}
Votre Bundle fera appel aux entités du bundle Helium.
